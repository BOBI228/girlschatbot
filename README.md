## Telegram компаньон-бот «Лина»

Дружелюбный и эмпатичный ИИ-бот для Telegram. Поддерживает беседу, помогает выговориться и допускает лёгкий, tasteful флирт без откровенного контента.

### Возможности
- Эмпатичное общение, короткие и тёплые ответы
- Ненавязчивый флирт (PG-13), без пошлостей
- Базовая память контекста диалога
- Безопасность: age gate (18+), отказ от NSFW, кризисные подсказки

### Быстрый старт
1. Установите зависимости:
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```
2. Создайте `.env` на основе `.env.example` и заполните:
   - `TELEGRAM_BOT_TOKEN` — токен от `@BotFather`
   - `OPENAI_API_KEY` — ключ OpenAI
3. Запуск бота:
```bash
python main.py
```

### Настройки
- Модели по умолчанию: `gpt-4o-mini` для чата, `omni-moderation-latest` для модерации

### Примечания по безопасности и этике
- Бот общается только со взрослыми (18+)
- Никакого откровенного сексуального или незаконного контента
- При признаках кризиса даёт поддержку и контакты служб помощи

### Структура
```
app/
  ai_provider.py   # клиент OpenAI (async)
  bot.py           # Telegram-бот и хэндлеры
  config.py        # настройки из .env
  memory.py        # короткая память диалога
  persona.py       # системный промпт персоны
  safety.py        # модерация и кризисные ответы
```
